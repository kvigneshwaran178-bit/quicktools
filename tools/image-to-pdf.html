How Image to PDF Conversion Works

Our converter uses the jsPDF library running entirely in your browser. Each image is rendered onto a PDF page with your chosen settings:

Available Width = Page Width - (Left Margin + Right Margin)
Available Height = Page Height - (Top Margin + Bottom Margin)

Scale = min(Available Width / Image Width, Available Height / Image Height)

Final Width = Image Width × Scale
Final Height = Image Height × Scale

X Offset = Left Margin + (Available Width - Final Width) / 2
Y Offset = Top Margin + (Available Height - Final Height) / 2

This ensures your image is centered and fits perfectly within the selected page and margin settings. The Fill mode scales the image to cover the entire page, and Original Size preserves the pixel dimensions of the source image.

All images are encoded as base64 data URLs in memory — nothing is sent to any external server at any stage.
